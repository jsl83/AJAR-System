What is AJAR?

AJAR is an application designed to help researchers find relevant research papers as they are published automatically using machine learning. Publications are collected daily using the Springer Nature API and compared with a user's favorited articles to assess similarity and provide recommendations. For more information about the Springer API, please visit their website below.

Springer Nature API Portal

Article Search Parameters

AJAR supports the following keywords when conducting article searches:

  • Date: returns articles published on the specified date
  • From: returns articles published on or after the specified date
  • To: returns articles published before or on the specified date
  • Journal: returns articles published in the specified journal

Date parameters are in YYYY-MM-DD form. To use a search keyword, include keyword:<parameter> in your query (no spaces). For example,

        "lung cancer from:2020-08-01 to:2020-08-31"

will find all articles including both lung and cancer published in August 2020.
